# Mohammad Samsad Hussain - Digital Resume

A professional digital resume website for Mohammad Samsad Hussain, Database Administrator & Solution Architect.

## Features
- Fully responsive design
- Modern, professional UI
- Interactive elements and animations
- Contact form
- Downloadable PDF resume
- Optimized for all devices

## Customization

### Update Personal Information
1. Replace profile picture in `images/profile.jpg`
2. Update contact information in `index.html`
3. Modify experience details as needed
4. Add your actual PDF resume file

### Styling
- Colors can be modified in the `:root` variables in `styles.css`
- Fonts and spacing can be adjusted as needed

## Deployment
1. Upload all files to GitHub repository
2. Enable GitHub Pages in repository settings
3. Your resume will be live at: `username.github.io/repository-name`
