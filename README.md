# StudyFlash Website

A Quizlet-inspired study platform website built with HTML, CSS, and JavaScript.

## Features
- Responsive design
- Interactive flashcard demo
- Modern UI with smooth animations
- Mobile-friendly layout

## Setup Instructions
1. Create a new GitHub repository
2. Upload all these files to your repository
3. Enable GitHub Pages in repository settings
4. Add your custom domain in GitHub Pages settings
5. Update DNS records at your domain registrar

## File Structure
- `index.html` - Main website page
- `styles.css` - All styling and responsive design
- `script.js` - Interactive functionality
- `README.md` - This documentation

## Customization
- Change colors in `styles.css`
- Update content in `index.html`
- Modify interactive features in `script.js`
